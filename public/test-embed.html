This page demonstrates how to embed forms from AnyForm into your website.

üìù Instructions:

  1. Create a form at http://localhost:3000 (or your deployment URL)
  2. Get the form ID from the share link (e.g., /f/YOUR_FORM_ID)
  3. Replace YOUR_FORM_ID in the iframe src below
  4. Customize the iframe dimensions and parameters as needed

Basic Embed Code:

<iframe src="http://localhost:3000/embed/YOUR_FORM_ID" width="100%" height="800" frameborder="0" style="border: none; border-radius: 8px;" ></iframe>

Advanced Embed with Options:

<iframe src="http://localhost:3000/embed/YOUR_FORM_ID?transparent=true&hideTitle=true&theme=light" width="100%" height="800" frameborder="0" style="border: none;" ></iframe>

Available Parameters:

  • transparent=true - Remove card background for seamless integration
  • hideTitle=true - Hide the form title
  • hideBranding=true - Hide the "Powered by AnyForm" footer
  • theme=light|dark|auto - Set the color theme
  • padding=16 - Adjust padding (in pixels)
